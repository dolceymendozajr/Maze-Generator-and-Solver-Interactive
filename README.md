# C++ Maze Generator and Solver Interactive

A C++ implementation of a NxN maze generation and solving (either by an algorithm or by user).
  
## Usage:
```
g++ maze_main.cpp Maze.cpp Graph.cpp
```
Execute a.out (Linux) or a.exe (Windows)
